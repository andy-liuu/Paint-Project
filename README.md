# Paint-Project
### About
Simple themed raster graphics editor made with Python3 and Pygame

### Theme
- Calvin and Hobbes
- Lo-fi

### Purpose
2018-2019 midterm project for ICS3U @ Vincent Massey S.S

### Features
- Color wheel
- Oval and Rectangle shapes
- Polygon shapes
- Pencil and Brush
- Spray can effect
- Hilighter
- Bucket tool
- Themed Stickers
- Themed Backgrounds
- Load and save as PNG/JPG
- Undo/Redo
- Play/Pause music
